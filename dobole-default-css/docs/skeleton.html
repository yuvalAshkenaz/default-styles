Skeleton / Loading States - Documentation

Skeletons are placeholders shown while content is loading. They improve perceived performance and give users a visual cue that something is happening.

<div class="skeleton skeleton-avatar"></div>

Notes

  • Skeletons can be used for text, images, cards, avatars, or any content container.
  • Add aria-busy="true" on parent containers during loading for accessibility.
  • Custom widths and heights can be applied via inline styles or additional classes.
