Accessible Alerts - Documentation

The Accessible Alert component is inspired by SweetAlert. It creates a modal message dynamically, with focus-trap, keyboard support, overlay click, and Promise-based result.

Demo Buttons

Usage

After importing initAlerts(), call it once. Then use the global showAlert(options) function.

JavaScript

import { initAlerts } from '../js/alerts.js';

document.addEventListener('DOMContentLoaded', () => {
  initAlerts();

  document.getElementById('successAlertBtn').addEventListener('click', () => {
    showAlert({ title: 'Success!', text: 'Operation completed successfully', icon: 'success', confirmText: 'OK' });
  });

  document.getElementById('warningAlertBtn').addEventListener('click', () => {
    showAlert({ title: 'Warning!', text: 'Be careful with this action', icon: 'warning', confirmText: 'Understood' });
  });

  document.getElementById('errorAlertBtn').addEventListener('click', () => {
    showAlert({ title: 'Error!', text: 'Something went wrong', icon: 'error', confirmText: 'OK' });
  });

  document.getElementById('confirmAlertBtn').addEventListener('click', async () => {
    const result = await showAlert({
      title: 'Are you sure?',
      text: 'This action cannot be undone',
      icon: 'warning',
      confirmText: 'Yes',
      cancelText: 'Cancel'
    });
    console.log('User choice:', result);
  });
});

Example Call

showAlert({
  title: 'Success!',
  text: 'Your changes have been saved.',
  icon: 'success',
  confirmText: 'OK'
}).then(result => {
  if(result) {
    console.log('User clicked OK');
  }
});

HTML Generated Dynamically

No static HTML is required for each alert. The alert modal is created by JavaScript when showAlert() is called.

Features

  • Dynamic creation, no pre-existing HTML needed
  • Trap focus inside the modal
  • Close with ESC key or clicking on the backdrop
  • Promise-based return value to handle user choice
  • Customizable title, text, icon, confirm and cancel buttons

Styling

Use CSS to style the modal and icons. Example classes:

.alert-backdrop { /* overlay background */ }
.alert-dialog { /* modal box */ }
.alert-header { /* title and optional icon */ }
.alert-body { /* message text */ }
.alert-footer { /* buttons container */ }
.alert-success, .alert-warning, .alert-error { /* icon colors */ }
