Dooble UI – Documentation

Welcome to the documentation for Dooble UI, a modular and responsive UI framework built with SCSS, designed for rapid development of modern, accessible interfaces.

Project Overview

Dooble UI contains:

  • Components: Buttons, Cards, Loaders, Tooltips, Mega Menus, Forms, etc.
  • Layout Helpers: Grid System, Containers, Flex Utilities, Spacing Helpers.
  • Data Visualization: Counters, Charts, Animated Numbers, Progress Bars.
  • Theme & Variables: Color maps, breakpoints, container widths, border-radius, shadows, etc.
  • Accessibility: ARIA attributes, keyboard navigation, RTL support.

How to Use

1. Include the compiled CSS file in your HTML:

<link rel="stylesheet" href="dobole-default-css.css">

2. Include the bundled JS for components (tooltip, mega menu, etc.):

<script src="js/bundle.js"></script>

3. Use the SCSS partials to customize your theme or add new components.

Best Practices

  • Use variables for colors, spacing, and breakpoints for consistent design.
  • Prefer SCSS partials for customization over direct CSS edits.
  • Keep components modular and reusable.
  • Follow accessibility guidelines for interactive elements.
  • Test your UI on multiple screen sizes and RTL direction if needed.
