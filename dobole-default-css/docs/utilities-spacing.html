Spacing & Display Utilities

Part of Dooble UI — a utility system inspired by Bootstrap 5.3, auto-generated from SCSS mixins and your project breakpoints.

Spacing Utilities

Use margin (m) and padding (p) classes to control spacing. Combine with direction (t, b, x, y) and breakpoint suffixes.

.m-3
.p-4
.mt-5 .mb-2
.mx-3-960 (applies from 960px)
<div class="m-3">...</div>
<div class="p-4">...</div>
<div class="mt-5 mb-2">...</div>
<div class="mx-3-960">...</div>

Border Radius Utilities

Control corner rounding with .rounded classes. Options include .rounded-0 to .rounded-4, as well as .rounded-top, .rounded-pill and more.

.rounded
.rounded-3
.rounded-pill
.rounded-top
<div class="rounded">...</div>
<div class="rounded-3">...</div>
<div class="rounded-pill">...</div>
<div class="rounded-top">...</div>

Display Utilities

Toggle visibility with .d-* classes. Works responsively with your defined breakpoints.

.d-block
.d-inline-block
.d-flex
<div class="d-block">...</div>
<div class="d-inline-block">...</div>
<div class="d-flex justify-content-center align-items-center">...</div>

Responsive Modifiers

Apply breakpoints to any utility class (e.g. -960, -1140, -1460) to make them responsive.

.d-flex-960 (applies flex from 960px) 
<div class="d-block d-flex-960 justify-content-center">...</div>

Summary

  • Spacing: m / p + direction + size + optional breakpoint (e.g. mt-3-960)
  • Radius: rounded-[0–4], rounded-circle, rounded-pill
  • Display: d-[block|flex|none] + breakpoint
  • Breakpoints: 540, 720, 960, 1140, 1460, 1920
