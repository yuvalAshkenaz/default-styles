Tooltips – Documentation

The tooltip component displays additional information when the user hovers or focuses on an element.

Examples

HTML Code

<button class="btn" data-tooltip="Tooltip from the top" data-tooltip-placement="top">Top</button>
<button class="btn" data-tooltip="Tooltip from the bottom" data-tooltip-placement="bottom">Bottom</button>
<button class="btn" data-tooltip="Tooltip at the start of the element" data-tooltip-placement="start">Start</button>
<button class="btn" data-tooltip="Tooltip at the end of the element" data-tooltip-placement="end">End</button>

Using JavaScript

Import the initTooltips() function and call it after the document has loaded: 

import { initTooltips } from '../js/tooltips.js';

document.addEventListener('DOMContentLoaded', () => {
  initTooltips();
});
